## Formulario

### Datos Clínicos
- Edad (años): 18-100
- IMC (kg/m²): 15-60

### Datos Histopatológicos
- Grado Histológico: Grado 1 (Bajo) / Grado 2 (Alto)
- Tamaño Tumoral (cm): 0-20
- Infiltración Miometrial: Sin infiltración / <50% / ≥50% / Serosa
- LVSI: No / Sí
- Infiltración Cervical: No / Glandular / Estroma
- Estadio FIGO 2023: IA, IB, IC, II, IIIA, IIIB, IIIC1, IIIC2, IVA, IVB

### Datos Moleculares
- p53 IHQ: Normal / Aberrante / No disponible
- Receptores Estrógenos (%): 0-100
- Receptores Progesterona (%): 0-100
